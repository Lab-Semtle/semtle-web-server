## 개발 환경

### 사전 준비

- `python` 버전 관리 목적 `pyenv` 설치
- `python` 패키지 관리 목적 `poetry` 설치

### 개발 빌드

1. 가상 환경 생성

   ```shell
   python -m venv .venv
   ```

2. 의존성 설치

   ```shell
   poetry install
   ```

3. 개발
   - IDE `vscode`로 개발
